¿Qué es RataLab?

RataLab es una plataforma web innovadora desarrollada por estudiantes de la Universidad Tecnológica Nacional - Facultad Regional Santa Fe (UTN-FRSF) para el control, monitoreo y análisis de robots móviles autónomos especializados en la resolución de laberintos.

Este proyecto integra múltiples disciplinas de la ingeniería electrónica, combinando programación de sistemas embebidos, comunicaciones IoT, algoritmos de optimización y desarrollo web para crear una solución completa de robótica educativa.

El sistema permite la interacción en tiempo real con robots basados en ESP32, visualización de trayectorias, control manual mediante interfaz web o teclado, y la aplicación de algoritmos avanzados de pathfinding para encontrar rutas óptimas en laberintos complejos.

Tecnologías Utilizadas

Hardware: Placa de desarrollo ESP32 con conectividad WiFi integrada, sensores ultrasónicos HC-SR04 para detección de obstáculos, sensores infrarrojos para seguimiento de línea, encoders para odometría precisa y motores DC con driver L298N.

Comunicación: Protocolo MQTT (Message Queuing Telemetry Transport), un protocolo ultraligero de mensajería publish/subscribe ideal para aplicaciones IoT. Utilizamos HiveMQ Cloud como broker para garantizar comunicación confiable y de baja latencia.
